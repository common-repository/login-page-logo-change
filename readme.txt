== Description ==
  
<p><strong>Login Page Logo Change</strong> - Allow you to add a custom logo to your WordPress login page instead of the default WordPress logo and also allows you to customize the height, width, margin, and padding of the logo.</p>
<p>By default login page has a WordPress logo so when you add your custom logo to your login page, it will help to enhance your branding.</p>
<br>
<p><strong>Features include:</strong></p>
 
<ul>
  <li>Add your custom logo to the login page by uploading image from your local storage</li>
  <li>Specify your logo image width in pixel</li>
  <li>Specify your logo image height in pixel</li>
  <li>Specify your logo image margin in pixel (Top, Left, Right, and Bottom)</li>
  <li>Specify your logo image padding in pixel (Top, Left, Right, and Bottom)</li>
</ul>  
  
== Installation ==
  
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
  
== Frequently Asked Questions ==
  
= How do I use this plugin? =

  You can use this plugin by uploading custom image and set height and width of the image from admin side

= How to uninstall the plugin? =

Simply deactivate and delete the plugin.
